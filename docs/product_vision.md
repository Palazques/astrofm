# Product: Astro.fm Vision and Core Identity

## I. Executive Summary and Vision
Astro.FM merges celestial calculations with music streaming to create personalized "sound profiles" based on users' birth charts and current planetary transits.
* **Core Goal:** Transform celestial movements into personalized sonic experiences.
* **Value Proposition:** Personalized Cosmic Audio, Daily Sonic Horoscopes, Data Sonification, Cosmic Frequency Matching.

## II. Product Tone and Aesthetic
* **Tone:** Cosmic, Spiritual, Modern Minimalism, and technically precise.
* **Vibe:** "Cosmic Nightlife" aesthetic.
* **UI Design:** Dark themes with **electric yellow and hot pink accents**. Uses **Glassmorphism UI** with geometric typography.

## III. Core Astrological Logic and Output
The app's logic translates planetary data into audio parameters.
* **Horoscope Output:** AI-generated horoscopes with **sonic metaphors** that translate into playlist parameters.
* **Data Sonification Engine:** Converts birth charts and transits into unique **sound signatures (Hz frequencies)** for meditation and frequency alignment.
* **Compatibility:** Features "Chart Syncing" and "Frequency Matching" for social compatibility analysis.

## IV. Key Features
| Category | Key Feature |
| :--- | :--- |
| **Calculation** | Birth Chart Calculator (Natal Chart), Daily Transit Engine, Advanced Transit Analysis (Premium). |
| **Audio Generation** | Data Sonification Engine, Daily Cosmic Frequencies, Meditation Sound Modes, Real-Time Audio Synthesis, Binaural Beat Integration. |
| **Curation** | Sonic Horoscope Generator, Automatic Playlist Generation, Harmonic Compatibility Analysis. |
| **Integration** | Multi-Platform Streaming: Spotify, Apple Music, YouTube Music. |
| **Social** | Chart Syncing, Frequency Matching, Synchronized Meditation, Daily Frequency Circles. |

## V. Technical Architecture Directives
* **Frontend:** Flutter Framework (iOS, Android, Web), responsible for UI/UX and State Management.
* **Backend:** **Flask API** (Python), responsible for all calculations and data processing.
* **Astrology Engine:** **Swiss Ephemeris** for accurate planetary positions.
* **House System:** **Whole Sign Houses** (simplified 30-degree divisions) for musical mapping.
* **Database/Auth:** Firebase Integration (Authentication, data storage, cloud functions).
